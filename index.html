Automated Radiology Report Generation for Tuberculosis CXRs

Developing a pipeline that combines bounding box annotations from radiologists with large vision-language models (GPT-4o, CheXagent, LLaVA-Med) to generate MIMIC-CXR style radiology reports for TB-positive chest X-rays from the VinDr and Montgomery datasets.

VinDr GPT-4o CheXagent LLaVA-Med Report Generation

Tuberculosis Detection and Localization in Chest X-rays

Investigating object detection and weakly supervised localization methods for TB lesion detection across multiple datasets including TBX11K, Montgomery County, and Shenzhen Hospital CXR sets.
